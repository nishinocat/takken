# 宅建1問1答 合格支援アプリ

## 概要
宅建試験合格を目指す方のための1問1答形式の学習アプリです。

## 機能
- **出題モード**
  - 全体ランダム出題
  - 項目別出題（ランダム/順番）
  - 復習モード（間違えた問題）
  
- **学習サポート**
  - 詳細な解説付き
  - 学習時間の記録
  - 正答率の表示
  - 分野別の分析
  - 重点学習領域の提案

## 使い方
1. `index.html`をブラウザで開く
2. 出題モードを選択
3. 問題に○×で回答
4. 解説を確認して理解を深める
5. 分析機能で弱点を把握

## デプロイ
VercelまたはGitHub Pagesで簡単にデプロイ可能です。

## 技術仕様
- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorageによるデータ保存
